Nearest-neighbor delivery

In this project I use Python and OOP The program will take in data, create a self-adjusting hash table, send out two trucks to deliver (with time & load constraints), and utilize the Nearest-neighbor algorithm for optimal calculation of delivery route.
Download the zip file, initiate main.py and interact via terminal
